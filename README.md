# API server for [Pokedex client](https://github.com/Ohar/pokedex)

## Install

1. Install Node.js 8.6.0+
2. Install Redis
3. `yarn install`
4. create `config.json` [as described](#config_json_example)

```bash
node index.js
```

## Usage

```bash
node index.js
```

## <a name="config_json_example"></a>`config.json` example

```json
{
  "port": 4242
}
```
